# Stilometri-Alquran-dan-Kitab-Suci-Lainnya
Hasil audit stilometri Alquran dan beberapa kitab suci lainnya

Repositori ini berisi ilustrasi grafis hasil audit beberapa kitab suci. 

## 📁 Isi Repositori
*   `stilometri alquran dan kitab suci lainnya.pdf` : Dokumen utama (10 halaman).
*   `README.md` : Dokumentasi panduan ini.

## 🔍 Cakupan Materi (10 Halaman)
Secara garis besar, file PDF di dalam repositori ini
1.  **Kitab suci sebagai baris kode**: Analisis kuantitatif gaya bahasa oleh mesin sehingga bebas bias manusia.
2.  **Metrik Audit**: Pendekatan 3 metrik pengukuran yaitu sidik jari teks, git diff (intervens), dan arsitektur data
3.  **Hasil audit**: 3 halaman hasil pengukuran dalam bentuk grafis.
4.  **Anomali Struktur Data Alquran**: Struktur Alquran mematahkan model arsitektur sekuensial linear manusia karena bersifat self-referensial, sistem cheksum terkunci, dan mustahil diurai atau diedit sebagai narasi linear tanpa merusak integritas seluruh model datanya.

## 🛠️ Cara Menggunakan

2. Pelajari setiap halamannya.
3. File dalam bentuk ilustrasi grafis, tidak mudah di-copy paste teks-nya tanpa aplikasi khusus.
